## **1ï¸âƒ£ Variables in JavaScript**
In JavaScript, variables can be declared using:

```javascript
var x = 10; // Old way (not recommended)
let y = 20; // Modern way (preferred)
const z = 30; // Constant value (cannot be reassigned)
```

### **ğŸ’¡ Key Differences**
- `var` has function scope.
- `let` has block scope and can be reassigned.
- `const` has block scope and **cannot** be reassigned.

## **2ï¸âƒ£ JavaScript Data Types**
JavaScript has **two main categories** of data types:

### **ğŸ”¹ Primitive Data Types** (Stored by value)
1. `String` â†’ Text data
   ```javascript
   let name = "Alice";
   ```
2. `Number` â†’ Numeric values
   ```javascript
   let age = 25;
   ```
3. `Boolean` â†’ `true` or `false`
   ```javascript
   let isLoggedIn = false;
   ```
4. `Undefined` â†’ Variable declared but not assigned
   ```javascript
   let job;
   ```
5. `Null` â†’ Explicitly empty value
   ```javascript
   let car = null;
   ```
6. `Symbol` â†’ Unique identifier (Advanced topic)
   ```javascript
   let uniqueID = Symbol("id");
   ```
7. `BigInt` â†’ Large numbers beyond `Number` limits
   ```javascript
   let bigNumber = 12345678901234567890n;
   ```

### **ğŸ”¹ Complex Data Types** (Stored by reference)
1. `Object` â†’ Collection of properties
   ```javascript
   let person = {
       name: "Alice",
       age: 25
   };
   ```
2. `Array` â†’ Ordered list of values
   ```javascript
   let colors = ["red", "blue", "green"];
   ```
3. `Function` â†’ Reusable block of code
   ```javascript
   function greet() {
       console.log("Hello, world!");
   }
   // verbalized way
   const x = function(){
      
       console.log("Hello, world!");
   }
   // with es6
   const greet = () => {
      
       console.log("Hello, world!");
   }
   ```
4. `Set` â†’ Collection of unique values
   ```javascript
   let uniqueNumbers = new Set([1, 2, 3, 3, 4]);
   ```
5. `Map` â†’ Key-value pairs
   ```javascript
   let userMap = new Map();
   userMap.set("name", "Alice");
   userMap.set("age", 25);
   ```

## **3ï¸âƒ£ Looping Through Complex Data Types**

### **ğŸ”¹ Looping through Arrays**
```javascript
let colors = ["red", "blue", "green"];
for (let color of colors) {
    console.log(color);
}
```

### **ğŸ”¹ Looping through Objects**
```javascript
let person = { name: "Alice", age: 25 };
for (let key in person) {
    console.log(key + ": " + person[key]);
}
```

### **ğŸ”¹ Looping through Sets**
```javascript
let uniqueNumbers = new Set([1, 2, 3, 4]);
for (let num of uniqueNumbers) {
    console.log(num);
}
```

### **ğŸ”¹ Looping through Maps**
```javascript
let userMap = new Map();
userMap.set("name", "Alice");
userMap.set("age", 25);

for (let [key, value] of userMap) {
    console.log(key + ": " + value);
}
```
